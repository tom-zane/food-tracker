# MacroTrack

MacroTrack is a modern nutrition and fitness tracking app built with React Native and Expo.

Track calories, macros, body weight, workouts, and daily progress with a clean, fast, and premium mobile experience.

---

## Features

- Calorie tracking
- Macro tracking
- Weight tracking
- Progress graphs
- Nutrition logging
- Barcode scanning support
- Dark modern UI
- Fast and lightweight experience
- Offline-friendly architecture
